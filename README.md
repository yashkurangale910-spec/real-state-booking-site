# 🏡 Nestly - Real Estate Booking Platform

Nestly is a premium, high-end real estate booking platform designed with a focus on cinematic UI/UX and seamless full-stack integration. It allows users to browse luxury properties with advanced filtering and a smooth, interactive experience.

## ✨ Key Features

- **Cinematic UI/UX**: Implemented with React and custom Vanilla CSS for a premium feel.
- **Dynamic Property Listings**: Real-time property fetching from a Spring Boot backend.
- **Advanced Filtering**: Quickly find properties based on location, price, and category.
- **Responsive Design**: Optimized for all devices, from desktop to mobile.
- **Interactive Map Integration**: (Planned) Visualizing property locations for better context.

## 🛠️ Tech Stack

### Frontend
- **React**: Component-based UI library.
- **Vite**: Ultra-fast build tool and dev server.
- **Vanilla CSS**: Custom design system for maximum flexibility and performance.
- **Framer Motion**: (Planned/Integrated) High-end animations and transitions.

### Backend
- **Spring Boot**: Robust Java-based backend framework.
- **RESTful API**: Clean and documented endpoints for property data.
- **Maven**: Dependency management and build automation.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- JDK 17+
- Maven

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yashkurangale910-spec/real-state-booking-site.git
   ```

2. **Frontend Setup**:
   ```bash
   cd nestly-ui
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd nestly-api
   mvn clean install
   mvn spring-boot:run
   ```
